# HR System - AI Agent Instructions

File referensi untuk AI agents (Cursor, Claude, GitHub Copilot, Gemini). Detail ada di file masing-masing platform.

## Quick Reference

| Platform  | File |
|-----------|------|
| Cursor    | `.cursor/rules/*.mdc` |
| Claude    | `CLAUDE.md`, `.claude/rules/*.md` |
| GitHub    | `.github/copilot-instructions.md` |
| Gemini    | `GEMINI.md` |

## Project

Ruby on Rails 7.2 HR System. Tech: Ruby 3.3, Stimulus, Chart.js. UI: Indonesia.

## Commands

```bash
bundle install && bin/rails db:migrate db:seed
bin/rails server
bin/rails test
```
